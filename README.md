## 🚀 Key Features

* **Interactive 3D Galaxy:** A custom-built WebGL environment using **Three.js** that reacts to mouse movements and scroll depth.
* **✨ AI Project Scoper:** Integrated **Gemini AI** functionality that allows potential clients to draft project ideas and receive an instant technical stack suggestion.
* **Wali's AI Agent:** A floating chat widget powered by generative AI to answer questions about my skills and experience.
* **Glassmorphism UI:** Modern, clean aesthetics with blurred panels and high-contrast gradients.
* **Fully Responsive:** Optimized for everything from high-end desktops to mobile touch devices.
